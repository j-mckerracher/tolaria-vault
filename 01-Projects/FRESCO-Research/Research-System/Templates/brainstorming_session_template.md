## How to Use This Template

1. **Before session**: Fill in Session Info
2. **During session**: Run brainstorming prompt (see `brainstorming_prompt.md`)
3. **Copy LLM output**: Paste relevant parts into RQ sections
4. **Evaluate & synthesize**: Fill in Synthesis and Top 3 Ideas
5. **Plan next steps**: Create action items
6. **Archive**: Save this in a session folder in Planning/ or Logs/
