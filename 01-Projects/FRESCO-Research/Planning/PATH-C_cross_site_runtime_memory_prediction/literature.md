## Literature Gap

Large-scale, production HPC workload prediction work often reports strong within-site performance, but cross-site transfer is less consistently quantified with strict leakage avoidance and reproducible evaluation. This path will position its contribution as a transfer-matrix baseline (train site × test site) with confidence intervals, plus error archetype analysis and few-shot calibration.
